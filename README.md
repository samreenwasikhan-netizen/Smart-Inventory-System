## 🌟 Features

- **Real-Time KPI Tracking:** Summary metrics displaying total products, low-stock alerts, and out-of-stock counts.
- **Full CRUD Operations:** Add, edit, view, and delete product records connected directly to a SQL database.
- **Dynamic Search:** Filter inventory items dynamically by name or category without reloading the page.
- **Responsive Dark Mode UI:** Designed with modern CSS Grid and Flexbox for a clean, user-friendly experience.

---

## 🛠️ Tech Stack

- **Backend:** C# / ASP.NET Core Web API, Entity Framework Core
- **Database:** Microsoft SQL Server
- **Frontend:** HTML5, CSS3, JavaScript (Fetch API)
- **Tooling:** VS Code, Git, GitHub
